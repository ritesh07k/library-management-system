# Library Management System

This is a full-stack web application for managing library operations such as book listings, borrowing, and user management.  
It is built using Spring Boot for the backend and React.js with TypeScript for the frontend.

## Features
- Add, update, and view books
- Borrow and return tracking
- My
- SQL-based relational data storage
- Layered backend architecture
- Responsive frontend built with React and TypeScript

## Tech Stack
- Frontend: React.js, TypeScript, HTML, CSS  
- Backend: Spring Boot, REST APIs, Spring MVC  
- Database: MySQL  
- Tools: Maven, Git, Postman

## Architecture
Frontend (React + TypeScript) → Spring Boot REST APIs → MySQL Database
